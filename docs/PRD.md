# PRD

## Problem

Run retrospectives make long autonomous runs inspectable by preserving decisions, verification evidence, unresolved risks, and follow-up work without uploading private logs.

## Users

Agent builders, OSS maintainers, and operators who need local evidence before a run proceeds.

## MVP Scope

- Local-first CLI
- Library API
- Fixture-backed tests
- Smoke command
- Skill instructions

## Non-Goals

- Live connector writes
- Hosted service
- Package publishing
- Automatic approvals
